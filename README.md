### 🧭 Current Work
- Reinforcement learning for autonomous exploration using learned depth, semantics, metric-semantic maps, and 3D scene graphs.
- Real-time monocular RGB+IMU perception for lightweight robots, with focus on SLAM/VIO, dense prediction, mapping, and scene understanding.
- Releasing research code and reproducible tools around my PhD work at the University of Twente.

### 🔬 Research & Code
| Project | Focus | Status |
| --- | --- | --- |
| **Spatial RL for Exploration** | RL policies that use learned spatial representations for exploration and navigation. | Current focus |
| **Mono Hydra++** | Monocular scene-graph construction with multi-task perception for 3D indoor mapping. | Under review |
| **M2H-MX** | Real-time dense depth + semantic prediction integrated with monocular SLAM. | ICRA 2026 SRRA Workshop |
| **M2H** | Cross-task attention for depth, semantics, edges, and surface normals. | IROS 2025 |
| **mono_hydra** | Real-time 3D scene graphs from monocular camera + IMU. | ISPRS 2023 |
